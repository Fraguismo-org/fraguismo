**Context:** When checking the "Pró Liberdade" reinvindication, the superuser MUST be able to remove a picture

**Technical Constraints:** On "Aprovar Reinvidicacoes", when clicking on a request to moderate it, the pictures
added by the user will have a button to allow to delete it by administrator

* [ ] Button "Remover" next to the picture
* [ ] The admin clicks on "Remover", a window pops up asking for confirming the picuture removal
* [ ] After accepting removing picture, the picture must be fisically removed from the media, and database
* [ ] The picture no more present on the revindication, media folder, and database
